## SEEDHA BUILD KARO — 3 Steps

### Step 1 — Java 21 install karo
https://adoptium.net → Java 21 download karo
Check: `java -version` → "21" dikhna chahiye

### Step 2 — Build karo (ek command!)
Yeh ZIP extract karo, folder mein jao:

**Windows:**
```
gradlew.bat build
```
**Mac/Linux:**
```
chmod +x gradlew && ./gradlew build
```
Pehli baar 3-5 min lagenge (internet se files download karega)

### Step 3 — JAR lo, mods folder mein daalo
```
build/libs/rinkuswap-1.0.0.jar  →  .minecraft/mods/
```
Fabric API bhi chahiye: https://modrinth.com/mod/fabric-api

## Kaise kaam karta hai?

### STUN SLAM MODE ⚡ (Default)
```
1. Kisi height se giro
2. Girte waqt LEFT CLICK dabao
3. Mod INSTANTLY mace slot pe switch karta hai (0-tick)
4. Mace ka full SLAM/BREACH damage register hota hai
5. 1 tick baad AUTOMATICALLY waapis original slot
```

### NORMAL MODE
```
Left click dabao → Mace slot
Click chodo      → Waapis
```
